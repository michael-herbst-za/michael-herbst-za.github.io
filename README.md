# Michael Herbst - CV

A clean, single-page HTML curriculum vitae hosted on GitHub Pages.

**Live site:** [https://michael-herbst-za.github.io](https://michael-herbst-za.github.io)

## About this repository

- `index.html` - The full CV page. Self-contained: no build step, no external dependencies.
- Designed for responsive viewing and clean print-to-PDF output.

## Updating the CV

1. Edit `index.html` directly.
2. Commit and push. GitHub Pages rebuilds automatically (usually within a minute).
3. Verify the live page, then use the browser's **Print → Save as PDF** to produce an A4 document if needed.
